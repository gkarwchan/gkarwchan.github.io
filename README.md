# BlogWebsite #
This is the code that generate the content of my personal blog : https://blog.gisspan.com.  


## Technology used ##
This code use [Jekyll static site generator](https://jekyllrb.com/).

## License and usage ##
Feel free to use the code here, to build your own blog

## Installation ##
You need ruby to run this code. After installing ruby, run the following command from the code directory.  

bundle install
